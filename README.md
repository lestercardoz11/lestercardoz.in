# Lester Cardoz - v2

This repository contains my personal portfolio, built with Next.js and Chakra UI. It showcases my work, skills, and provides a way for visitors to connect with me.

## Features

- **Responsive Design:** The portfolio is fully responsive and accessible on all devices.
- **Dark Mode:** A theme toggle allows users to switch between light and dark modes.
- **Animations:** Subtle animations and transitions are used to enhance the user experience.
- **SEO Optimized:** The portfolio is optimized for search engines with a sitemap and robots.txt.

## Technologies Used

- [Next.js](https://nextjs.org/) - A React framework for building server-side rendered and statically generated web applications.
- [Chakra UI](https://chakra-ui.com/) - A simple, modular, and accessible component library for React.
- [Framer Motion](https://www.framer.com/motion/) - A library for creating animations in React.
- [Vercel](https://vercel.com) - A cloud platform for static sites and Serverless Functions.

## Installation & Set Up

1. **Clone the repository:**

   ```sh
   git clone https://github.com/lestercardoz11/lestercardoz.com.git
   ```

2. **Navigate to the project directory:**

   ```sh
   cd lestercardoz.com
   ```

3. **Install dependencies:**

   ```sh
   yarn install
   ```

4. **Create a `.env.local` file:**

   Duplicate the `.env.example` file and rename it to `.env.local`. This file is used for environment variables.

   ```sh
   cp .env.example .env.local
   ```

5. **Start the development server:**

   ```sh
   yarn dev
   ```

   The application will be available at `http://localhost:3000`.

## Deployment

This portfolio is deployed on [Vercel](https://vercel.com). The site is automatically rebuilt and deployed when changes are pushed to the `main` branch.
